# 🌍 WanderLust

🚀 **WanderLust** is a travel exploration app that helps users **discover destinations and manage trips**.  

## 🔧 Tech Stack  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Atlas)  

## 🚀 Installation  
```bash
git clone https://github.com/arunby15/WanderLust.git
cd WanderLust
npm install
npm start
